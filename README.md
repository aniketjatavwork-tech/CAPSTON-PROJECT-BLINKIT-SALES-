## 📌 Key Insights  
- 💰 **Total Sales:** $1.20M+  
- 📦 **No of Items:** 9K+  
- ⭐ **Average Rating:** 3.9 – 4.0  
- 🏬 **Outlet Types & Locations:** Comparative performance by Tier & Size  
- 🍞 **Item Categories:** Breads, Breakfast, Canned, Dairy, Frozen, Household & more  
- 🥗 **Fat Content:** Low Fat vs Regular contribution
